**Module Federation:**
- Each micro-frontend is independently built and deployed
- Shell application loads modules dynamically at runtime
- Shared dependencies (Angular, RxJS, ZUILibrary) loaded once
- Teams can deploy their module without coordinating with other teams

### ZUILibrary (Component Library)

The shared component library provides consistent UI:

- **50+ components** — Buttons, forms, tables, charts, modals, toasts
- **Design tokens** — Colors, spacing, typography from design system
- **Accessibility** — WCAG 2.1 AA compliance
- **Storybook** — Interactive component documentation
- **Angular schematics** — Code generators for common patterns

### State Management (NgRx)

- **Store** — Global state tree per micro-frontend
- **Actions** — Events that trigger state changes
- **Reducers** — Pure functions that compute new state
- **Effects** — Side effects (API calls, navigation)
- **Selectors** — Memoized state queries for components

**Gateway Responsibilities:**
- **Authentication** — Validate JWT tokens from Auth0
- **Authorization** — Check RBAC permissions per endpoint
- **Rate limiting** — Per-user and per-org limits
- **Request routing** — Route to correct microservice based on path
- **Response caching** — Cache idempotent GET responses
- **Request logging** — Structured audit logs for compliance

### BFF (Backend for Frontend)

The BFF layer aggregates data from multiple microservices into frontend-optimized responses:

- Dashboard home page calls 5+ microservices — BFF combines into one response
- Reduces frontend round-trips (1 BFF call vs. 5 microservice calls)
- Handles data transformation for frontend consumption
- Implements frontend-specific business logic

### Performance Optimization

- **Lazy loading** — Micro-frontends loaded on-demand (route-based)
- **Tree shaking** — Unused code eliminated at build time
- **Ahead-of-Time (AOT) compilation** — Templates compiled at build time
- **Service Worker** — Caches static assets for offline-capable shell
- **Virtual scrolling** — Large lists (profiles, events) rendered on-demand
- **CDN** — Static assets served from Cloud CDN edge nodes
