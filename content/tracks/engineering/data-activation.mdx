## 🚀 Data Activation

Data Activation is the final mile — delivering audiences, events, and profile data to 150+ external platforms. This module covers the connector architecture, data transformation, delivery guarantees, and SmartPixel.

Each destination is a self-contained plugin implementing a standard interface:

- `authenticate()` — OAuth flow, API key validation
- `mapFields(profile) → destinationPayload` — Transform Zeotap data to destination format
- `deliver(payload)` — Send data to destination API
- `checkHealth()` — Verify connection is alive
- `getCapabilities()` — Declare supported features (batch, streaming, field types)

### Data Transformation

The transformation layer handles format differences between Zeotap and destinations:

- **Field mapping** — Map Zeotap attribute names to destination field names
- **Value transformation** — Convert data types (e.g., ISO date to epoch timestamp)
- **Hashing** — SHA-256 hash PII fields required by ad platforms
- **Filtering** — Only send consented data, only send changed fields
- **Batching** — Group records for bulk API calls

### Delivery Guarantees

**At-least-once delivery:**
- Every record is tracked with a delivery ID
- Failed deliveries retried with exponential backoff (1s, 2s, 4s, 8s, max 5 retries)
- Permanently failed records routed to DLQ with error details
- Delivery status tracked per record, per sync, per destination

**Rate Limiting:**
- Per-destination rate limits respected (e.g., Google Ads: 10 requests/second)
- Token bucket algorithm for smooth rate limiting
- Burst allowance for catching up after downtime

### SmartPixel

SmartPixel enables **real-time web personalization** without round-trip to the server:

1. SmartPixel JavaScript loaded on customer's website
2. On page load, fetches user's segment membership from edge cache
3. Executes personalization rules client-side (show/hide elements, swap content)
4. Latency: <50ms for personalization decision

**Use Cases:**
- Show personalized banner to "High Value" segment
- Hide promotion for users who already purchased
- Display loyalty tier-specific content
