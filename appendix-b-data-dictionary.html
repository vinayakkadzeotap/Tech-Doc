Complete field reference for all data models in the Zeotap CDP: profile attributes, identifier types, event schemas, consent fields, and AI-generated scores.

Identifier Types

Zeotap resolves users across 10 supported identifier types. All PII-containing identifiers are stored only as SHA-256 hashes.
