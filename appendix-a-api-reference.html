Profile API External

Source: Chapter 5 — Profile Store · Official docs: docs.zeotap.com

Customer-facing REST API to read, write, and delete Unified Customer Profiles. All endpoints use API Key authentication (apikey header). Available over HTTPS (https://api.zeotap.com) and mTLS (https://mtls-api.zeotap.com).

MethodEndpointOperationAuth
POST /cdp/v1/users/_search Fetch / Read user profile by identity value API Key
POST /cdp/v1/users Create or Update (upsert) user profile API Key
POST /cdp/v1/users/_delete Delete user profile(s) — triggers GDPR erasure pipeline API Key

POST /cdp/v1/users/_search — Fetch Profile

Returns profile attributes, IDs, consent, marketing preferences, and AI-computed attributes for a matched user. Supports up to 5 identity values per call.

Request

{
  "orgId": 1234,
  "regions": ["EU"],
  "fetch": {
    "ids":             ["AdId"],
    "profile":         ["age", "user_city"],
    "consent":         ["gdpr_advertising"],
    "mkt_preferences": ["telemarketing"],
    "calc_attributes": ["loyalty_score"]
  },
  "search": {
    "email_sha256_lowercase": [
      "9dff3928e6f62ef808"
    ]
  }
}

POST /cdp/v1/users — Upsert Profile

Creates or updates a profile. One identity per call. Returns 204 No Content on success.

// Request body
{
  "orgId": 1234,
  "region": "EU",
  "ucid": "d7a6bc75-2c87-4650-91dc-c212771866c9",
  "ids": {
    "email_sha256_lowercase": "0a0601cfc9d014580fcf0b0"
  },
  "upsert": {
    "profile":         { "user_gender": "M", "age": "34" },
    "mkt_preferences": { "telemarketing": "market", "sms_optin": "yes" },
    "consent":         { "gdpr_advertising": "yes" }
  }
}
// Response: 204 No Content

POST /cdp/v1/users/_delete — Delete Profile

Initiates deletion of one or more profiles. Triggers the full GDPR erasure pipeline (7–10 days for full purge). Max 400 IDs per call.

// Request body
{
  "orgId": 1539,
  "region": "EU",
  "ids": {
    "ucid": ["040ca427-21a2-4b8e-9f32-abc123def456"],
    "AdId": ["ddd682ec-bbe9-4705-bb02-3b9eacaf93bf"]
  },
  "sendEmailNotification": false
}
// Response: 200 OK
{
  "status": "accepted",
  "deletionRequestId": "del_8f3c2a1b-4d5e-6f78-9012-abcdef123456",
  "estimatedCompletionDays": "7-10"
}
